### Log4j에 관하여...  

로그레벨 우선순위는 다음과 같습니다. 
 - Fatal > Error > Warn > Info > Debug > Trace
 
 Log4j 구성은 크게 3가지가 있습니다.  
  - Logger: 메세지 로그를 남기는데 사용됩니다.  
  - Appenderr: 로깅정보를 어떤 형태로 남기는지 설정하는데 사용됩니다. (File, database, console etc..,)  
  - Layout: 로그를 남기는 형태/포맷을 정하는데 사용됩니다.
